# 🏥 MEDINOVA - Healthcare Website

A modern and responsive healthcare website built using **HTML, CSS, and Bootstrap**.
This project represents a complete medical landing page with multiple sections like services, doctors, blog, and appointment booking.

## ⚙️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/medinova.git
```

2. Open the project folder:

```bash
cd medinova
```

3. Run the project:

* Simply open `index.html` in your browser
